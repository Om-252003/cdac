1) write a function to accept a number and return its square using
	a) normal function
	b) lambda

a)	
def sqr(a):
	return a**2

print(sqr(10))

b)
print((lambda x:x**2)(10))



2) write a function to display "Hello World" using
	a) normal function
	b) lambda

a)	
def Hello():
	return "Hello World"

print(Hello())

b)
(lambda : print("Hello World"))()

3) write a function with 2 arguments , second argument should be "default argument" and display them. Using
	a) normal function 
	b) lambda

a)	
def disp(a, b = 10):
	return a , b

print(disp(a))

b)
(lambda a,b=20 : print(a,b))(10)

4) write a function with variable no. of arguments and display them. Using
	a) normal function
	b) lambda


a)	
def disp(*a):
	return [i for i in a]

print(disp(1,2,3))

b)
print((lambda *a: [i for i in a])(1,2,3) )
